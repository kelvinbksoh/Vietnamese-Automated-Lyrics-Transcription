# Installation
Install transformers, librosa and set GEMINI API KEY if you were to use LLM to correct errors and render

# Notebooks
- **`phoWhisper_demo.ipynb`**: Demo to chunk the audio and feed phoWhisper
- **`phoWhisper_test.ipynb`**: Function to chunk the audio and feed phoWhisper
- **`postprocessing_whisper.ipynb`**: Code to post process whisper output
- **`render_correction.ipynb`**: Notebook to use Gemini to error correct
